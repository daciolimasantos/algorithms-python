# Max Area of Island

## Problem

Given a grid of 0 and 1,
find the largest island.


## Idea

Use DFS to explore each island.
Count area recursively.


## Algorithm

1. Iterate grid
2. When find 1
3. Run DFS
4. Count area
5. Track max


## Complexity

Time: O(m * n)

Space: O(m * n)
